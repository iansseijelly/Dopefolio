Project Overview

Sensor timing characteristics critically impact the perfor- mance of cyberphysical systems. [4] Robot maximum speed is restricted by sensor/perception latency [7], and sensor- to-photon latency is a key metric in extended-reality (XR) systems. To deliver real-time-performance, state-of-the-art robotics and XR systems utilize a plethora of sensors as well as dedicated system-on-chips (SoCs) to process sensor data. To effectively design robotics SoCs, it is critical to build evaluation platforms that accurately model sensor behavior.
